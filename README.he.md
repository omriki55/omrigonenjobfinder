## מה זה

חברות משתמשות ב-AI כדי לסנן מועמדים. JobSearch AI נותן למועמדים AI כדי **לבחור חברות** — בלי להתקין כלום.

בניגוד לכלים מבוססי טרמינל, זו **אפליקציית ווב ללא התקנה**: בלי CLI, בלי npm, בלי קבצי קונפיג. סבתא שלך יכולה להירשם. כל המערכת רצה על אתר סטטי ב-GitHub Pages + Supabase + ‏GitHub Action אחד — הכול בחינם.

> **פילוסופיה: זה פילטר, לא תותח spray-and-pray.** ה-AI ממליץ להגיש רק כשההתאמה המשוקללת היא 6/10 ומעלה. הזמן שלך יקר — וגם של המגייסת.

## התחלה מהירה (משתמש)

1. פתחו את [דף הנחיתה](https://omriki55.github.io/omrigonenjobfinder/landing.html)
2. לחצו **להתחלה** ← בחרו שם משתמש וסיסמה
3. מלאו את האשף (או העלו קו"ח) והדביקו [מפתח Anthropic](https://console.anthropic.com)
4. זהו. המשרות יופיעו ב**המשרות שלי** אחרי הסריקה הבאה — אוטומטית.

זו כל ההתקנה. אין מה להתקין.

## ארכיטקטורה

ראו את התרשים ב-[README האנגלי](README.md#architecture). בקצרה: אתר סטטי (Pages) ↔ Supabase עם RLS ↔ GitHub Action שסורק ומדרג עם המפתח של כל משתמש.

רוצים אינסטנס משלכם עם משתמשים משלכם? ראו **[docs/SELF-HOST.md](docs/SELF-HOST.md)** — ‏Fork, פרויקט Supabase חינמי, קובץ SQL אחד, שני Secrets. בערך רבע שעה.
